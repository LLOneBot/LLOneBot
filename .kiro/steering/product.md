# Product Overview

LLTwoBot (Lucky Lillia Two Bot) is a QQ bot implementation that provides OneBot 11 and Satori protocol server capabilities for QQ messaging automation.

## Core Purpose

- Bridges QQ messaging with bot frameworks through standardized protocols (OneBot 11, Satori)
- Enables automated QQ interactions including messages, groups, friends, and file operations
- Provides web UI for configuration and management
- Supports multiple connection types: HTTP, WebSocket, and reverse connections

## Key Features

- Message handling (send/receive/recall) for private and group chats
- Group management (member operations, admin controls, notices)
- Friend management and requests
- File operations (upload/download, group files, flash files)
- Media processing (images, audio, video with FFmpeg)
- Event system for notifications and requests
- Multi-protocol support (OneBot 11, Satori)
- Web-based configuration interface

## Target Users

Bot developers and QQ automation users who need programmatic access to QQ functionality through standard bot protocols.
